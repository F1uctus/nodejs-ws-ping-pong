# Node.js WebSocket Ping-Pong

Bare-bones WebSocket ping-pong implementation with Node.js. For testing &amp; educational purposes only.

Hosted on https://nodejs-ws-ping-pong.herokuapp.com/

## Running

```bash
cd server && npm start
```
